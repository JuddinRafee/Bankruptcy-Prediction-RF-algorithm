What is BPS ?

BPS is stand for Bankruptcy Prediction System that have ability to predict whether a company or firm is about to bankrupt or not by accessing selected attributes of financial value .

What is RF ?

RF stand for Random Forest. The random forest is a classification algorithm consisting of many decisions trees. It uses bagging and feature randomness when building each individual tree to try to create an uncorrelated forest of trees whose prediction by committee is more accurate than that of any individual tree.

Save Time

BPS will predict if the companies is about to bankrupt or not by accessing the financial ratios that already been analysis and filter and give the prediction result immediately.

Reduce Work

BPS have include with machine learning model with high accuracy of prediction. All the analysis have been done in the model to make the best prediction. The result and financial ratios will be display in visual representation in order to make it more easier to understand the company condition and make immediate decision.
